# Deferred Features
Explicit scope decisions — things intentionally not being built yet.
Updated via the `pre-commit` skill. Loaded at every session start.

**Distinction from GDD Out of Scope:**
- **GDD Out of Scope** → permanent. "We will never build X."
- **This file** → temporary. "We are not building X until Phase N or until Y condition."

**The AI must not suggest, plan, or implement anything listed here
without the user explicitly promoting it first.**
